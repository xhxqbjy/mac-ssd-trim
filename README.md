## 先说问题：SSD 为什么越用越慢

这里要稍微解释一下底层原理，但别担心，不会太枯燥。

传统机械硬盘（HDD）删除数据的逻辑很简单：把那块区域标记成「可以覆盖」，下次写入时直接盖上去就行了。快，没有多余操作。

固态硬盘（SSD）就不一样了。它在写入新数据之前，**必须先把旧数据完整擦除一遍**，然后才能写进去。等于每次写文件都要多一道工序——先擦后写，同时进行，效率自然低下。时间一长，磨损不均衡，速度越来越慢，寿命也悄悄在缩短。

TRIM 指令的作用，就是把这个流程优化：**删除操作发生时，立刻清空对应数据块**，下次需要写入时直接写，不用现场擦。

苹果自家的原装 SSD 默认开启了 TRIM。但如果你给 Mac 换了第三方 SSD（比如自己升级了一块三星或西数的固态），系统不会自动开启这个功能——这就是问题所在。
